# GuessFace
A game to guess Algolia Faces

![Demo GIF](./demo.gif)

## Running

From the `guess_face` folder:
```
flutter run
```

## Building the release APK
From the `guess_face` folder:
```
flutter build apk --release
# Alternatively, to build an App Bundle:
flutter build appbundle --release
```
